# Documentacion del Sistema de Gestion Escolar

Esta carpeta contiene la documentacion funcional, tecnica y de producto del sistema, redactada como si el producto estuviera en fase de definicion inicial.

## Proposito

Servir como fuente unica de referencia para:

- Vision de producto y alcance
- Epicas e historias de usuario
- Requisitos funcionales y no funcionales
- Modelo de datos conceptual
- Arquitectura de alto nivel
- Diagramas de procesos y componentes
- Roadmap y criterios de evolucion

## Enfoque

La documentacion esta escrita en lenguaje de negocio y producto, evitando depender de decisiones de implementacion innecesarias. Cuando se mencionan capacidades tecnicas, se hace solo para dar contexto de la solucion futura.

## Notas de uso

- Cada documento puede evolucionar a medida que el proyecto avance.
- Las historias de usuario incluyen criterios de aceptacion para facilitar refinamiento y estimacion.
- Los diagramas estan expresados en Mermaid para mantener trazabilidad y versionado simple.

## Convenciones

- Prioridad: `Alta`, `Media`, `Baja`
- Estado: `Propuesto`, `En analisis`, `Aprobado`, `Implementado`
- Roles: `Administrador`, `Inspector`, `Docente`, `Asistente`, `Apoderado`

## Alcance del producto

El sistema esta concebido para administrar procesos escolares operativos, con foco en:

- Gestion de cursos y estudiantes
- Registro de asistencia y atrasos
- Control de salidas anticipadas
- Autenticacion y control de acceso
- Consulta operativa y paneles de seguimiento
